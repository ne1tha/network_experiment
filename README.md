# Routes Open Source Repo

这个目录用于整理未来开源版本的路线代码。

当前布局：

- `route1/`：路线1代码镜像
- `route2/`：路线2代码镜像
- `route3/`：路线3代码镜像
- `shared/`：未来可抽出的公共模块、统一脚本、公共文档占位

当前整理策略：

- 已从原工作目录复制路线1、路线2和路线3的代码骨架
- 默认排除了 `artifacts/`、`docs/`、`__pycache__/` 和 `.pyc`
- 暂时保留现有代码形态，不在这一轮做开源清洗

补充说明：

- `route3/` 这一轮额外做了最基本的开源清洗：
  - 删除了本地训练产物和本地 manifest
  - 只保留通用训练/评测脚本
  - 提供了公开示例配置与最小 README
  - 将明显写死的本机默认路径改成了仓库内相对约定或占位路径

后续开源前仍需处理：

- 清洗本地绝对路径和本地数据路径
- 统一依赖声明和环境安装方式
- 统一 README、许可证、示例配置和最小可运行入口
- 评估路线1和路线2是否需要抽出共享基础包
